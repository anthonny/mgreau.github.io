If you want to use the Sun14ReflectionProvider with XStream, you have to add a dependency to the "sun.jdk" module.

pass::[more]

[source,xml]
--
<?xml version="1.0" encoding="UTF-8"?>
<module xmlns="urn:jboss:module:1.0" name="mymodule">
  ...
  <dependencies>
    <module name="sun.jdk" />
  </dependencies>
</module>
--
