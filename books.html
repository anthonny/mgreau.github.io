Apache Maven

Maitrisez l'infrastructure d'un projet Java EE

Ce livre sur Apache Maven (en version 3 au moment de l’écriture), s’adresse à tout développeur amené à travailler sur des projets Java de taille conséquente. Il sert également de guide pour les architectes qui souhaitent mettre en œuvre Apache Maven sur un nouveau projet ou sur un projet existant. Apache Maven, créé il y a tout juste 10 ans, tient aujourd’hui une place centrale dans l’écosystème du développement logiciel sur la plate-forme Java EE.

Du développeur Java débutant à l’architecte technique, chaque lecteur trouvera dans cet ouvrage des informations claires et précises, illustrées par de nombreux schémas, pour aborder avec sérénité la découverte d’Apache Maven, jusqu’à sa gestion au sein d’une infrastructure projet. L’auteur propose régulièrement au lecteur de réaliser des travaux pratiques pour se forger une première expérience significative sur le produit.
